# mail-leyu-hub

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态文件形式存储，不依赖特定后端服务，可直接在浏览器中访问。仓库内容主要面向个人学习、资料整理及页面展示用途。

## 目录说明

```
├── index.html          # 仓库首页（可选）
├── pages/              # 存放独立 HTML 页面
│   ├── example1.html
│   └── example2.html
├── assets/             # 静态资源（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：主要存放各类 HTML 页面，每个页面均为独立功能单元。
- `assets/`：存放页面所需的通用静态资源，便于统一管理和引用。

## 页面归档说明

- 所有页面均为纯前端实现，无需后端支持。
- 页面内容可能涵盖学习笔记、实验原型、工具页面或信息展示等类型。
- 仓库不针对任何特定域名或网站，页面内容仅代表归档时的状态。
- 页面可能包含外部资源引用，使用时请确保网络连通性。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 目录下的任意 HTML 文件。
3. 也可通过 GitHub Pages 等静态托管服务部署后访问。
